# Operating Philosophy

> First do it, then do it right, then do it better.

## Phase 1: Do it ✓

Fork pi. Ship **ai** with bundled memory, telemetry, and play extensions.

## Phase 2: Do it right (current)

Consolidate under one name — **ai**.

- Single repo: `lyffseba/ai`
- Config namespace: `~/.ai/`
- Binaries: `ai` (default), `AI` (caps), `ai --fast` (nocaps helper)
- bet remains a **separate, untouched** repo — integrated via spawn only
- All decisions tracked in GitHub issues

## Phase 3: Do it better

- Rebrand workspace packages to `@lyffseba/ai-*`
- Regional leaderboard sync (opt-in)
- Opt-in telemetry → community-trained models
- Compete with OpenCode, Claude Code, and pi itself
