# ai extensions

Bundled on every session (disable with `ai -ne`).

| Extension | Purpose |
|-----------|---------|
| `mode.ts` | `ai` / `AI` / `--fast` prompts |
| `memory.ts` | `~/.ai/memory` + `/memory` |
| `telemetry.ts` | `~/.ai/telemetry` + `/telemetry` |
| `play.ts` | `/play` spawns [bet](https://github.com/lyffseba/bet) if installed |
